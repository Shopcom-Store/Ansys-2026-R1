Here is a concise overview of the key themes and updates across the portfolio.

🔬 Core Physics & Engineering Updates
Optics & Photonics (Zemax, Lumerical, Speos)

Zemax OpticStudio: Nested Elements Tolerancing (NEST) for realistic opto-mechanical assemblies and a new Requirements Editor to link specs directly to optimization.

Lumerical: Direct GPU meshing and new fabrication-aware workflows for CMOS image sensors (TCAD to FDTD). Nonlinear ring modulator models account for self-heating effects.

Speos: New Optical Design Exchange (ODX) transfers native lens data directly to CAD, eliminating geometry reconstruction.

Fluids (Fluent, Discovery, Rocky, FreeFlow)

Fluent: Expanded native GPU solving for batteries, combustion, and radiation. New adjoint solver capabilities via web interface for automated shape optimization.

Discovery: Real-time conjugate heat transfer (CHT) with Joule heating. Fluid virtual walls reduce memory usage by replacing thin solids with boundary conditions.

Rocky: Distributed GPU computing (beta) for large-scale particle simulations. New Mixing Index Calculator and intermolecular forces modules.

FreeFlow: Adaptive SPH resolution dynamically adjusts particle size to reduce computational load.

Electronics (HFSS, Maxwell, SIwave)

HFSS: New modern UI with NVIDIA cuDSS GPU acceleration (1.6x faster frequency sweeps).

HFSS-PI introduces wideband 3D power integrity for complex 3D ICs and packages.

SIwave: Up to 10x faster DCIR simulations for complex PCBs.

Component 2.0 (Beta): A single component definition for HFSS, Icepak, and Q3D to streamline multiphysics workflows.

Structures (Mechanical, LS-DYNA)

Mechanical: New direct morphing workflow reduces geometry preparation time. Resource prediction helps users determine when GPU acceleration is beneficial.

LS-DYNA: Enhanced battery thermal runaway simulations (radiation support, internal heat generation) and improved S-ALE (Structured Arbitrary Lagrangian-Eulerian) for blast/fluid-structure interaction.

🌐 Systems & Digital Twins

Ansys TwinAI: New fusion modeling and temporal fusion transformers to align simulation data with real-world sensor inputs.

Ansys CoSim: Distributed co-simulation product allowing different subsystem tools to run natively while exchanging data seamlessly.

AVxcelerate Sensors: GPU-accelerated multispectral engine for physically accurate camera and sensor simulation in NVIDIA Omniverse.
